# 💰 Expense Tracker

A modern Expense Tracker web application built using HTML, CSS, and JavaScript.  
It helps users manage their income, expenses, savings, and budget efficiently.

## 🚀 Live Demo

 https://adityadubey2211-prog.github.io/expence-flow/

## 📌 Features

- ✅ Add Income and Expense transactions
- ✅ Edit transactions
- ✅ Delete transactions
- ✅ Data persistence using LocalStorage
- ✅ Search transactions
- ✅ Sort transactions by date and amount
- ✅ Filter transactions by category
- ✅ Income, Expense and Savings calculation
- ✅ Budget tracking
- ✅ Saving goal management
- ✅ Expense visualization using Pie Chart
- ✅ Dark/Light theme toggle
- ✅ Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
- LocalStorage
